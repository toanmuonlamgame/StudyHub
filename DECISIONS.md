## 2026-07-09 - Collaboration Workflow
Decision: Use Git plus project memory files as the shared context between Codex, Antigravity, and the user.

Reason:
- Chat history is not a reliable source of truth across tools.
- Project files are visible to both tools.
- Git commits make changes reviewable and recoverable.

Rule:
- One AI should be the main code editor for a task.
- The other AI should review or plan unless explicitly assigned to edit.
