# StudyHub Project Memory

## Project Identity
- Project name: StudyHub
- Product type: mobile learning platform
- Goal: learning project and internship portfolio
- Current phase: foundation / planning

## Product Purpose
StudyHub is a mobile learning platform for users who want to organize and study from their own learning materials.

The product should focus on helping users upload or create their own question banks and study content. The developer should not pre-seed question data as the main content source.

## Confirmed Product Direction
- Users bring their own study content.
- StudyHub should support user-owned question banks / learning materials.
- The first version should stay simple and practical.
- The project should be useful both as a learning exercise and as an internship portfolio project.

## Confirmed Tech Stack
- Frontend: Flutter / Dart
- Backend: Node.js
- Database: PostgreSQL
